Frontend Roadmap

Frontend development is the development of graphical user interface of a website, through the use of HTML, CSS and JavaScript, so that users can view and interact with that website

  • HTML : Hyper Text Markup Language or HTML defines the skeleton of any web page. It gives the structure to the Websites

  • CSS : Cascading Style Sheets or CSS is the styling language used for styling the websites.

    1. Pure CSS

    2. Bootstrap

    3. Tailwind

    4. Sass preprocessor

  • JavaScript : also commonly known as JS is a Scripting Language understood by the Web Browsers. Chrome introduced V8 engine to run JavaScript in web browsers


    1. Vanilla JS (ES6)

    2. jQuery

    3. React

    4. Angular

    5. Vue

    6. npm

    7. yarn

    8. TypeScript
